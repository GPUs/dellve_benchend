Getting Started
===============

DELLve is distributed as a standard Python package via PyPi.

You can install DELLve using pip and setuptools in one step:

.. code-block:: bash

	pip install dellve

Alternatively, you can install DELLve from our GitHub repo:

.. code-block:: bash

	pip install git+https://github.com/dellve/dellve_benchend.git

Command Line Tools
==================

DELLve comes with a set of CLI tools that allow user to access its
functionality via command line.

.. program-output:: dellve --help

Graphical User Interface
========================

DELLve GUI is hosted as a separate 
`DELLveDash <https://github.com/dellve/dellve-dash>`_ project on GitHub. 

Please refer to the offial `DELLveDash <https://github.com/dellve/dellve-dash>`_
documentation for installation and usage instructions.
